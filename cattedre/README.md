## Ruolo delle cattedre nei progetti

I docenti non sono erogatori di contenuti ma **facilitatori di progetti**.
Ogni cattedra partecipa ai progetti interdisciplinari con il proprio apporto specifico.

Vedi il [catalogo progetti](../progetti/README.md) per le associazioni progetto ↔ cattedra.

---

## Come aggiungere una cattedra

1. Copia il file `cattedra-A041.md` come template
2. Aggiorna classe di concorso, discipline, competenze e ruoli
3. Apri una PR per revisione collegiale
