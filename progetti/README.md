# Catalogo Progetti

I progetti sono il **cuore del curricolo**.
I contenuti disciplinari non vengono pianificati a priori: emergono dai bisogni di ogni progetto.

---

## Come leggere il catalogo

Ogni progetto ha:
- Un **anno di riferimento** (o più, per i progetti pluriennali)
- Un **tema** e un **problema reale** da risolvere
- Le **cattedre coinvolte** e i relativi contributi
- Gli **strumenti** e le **tecnologie** usate
- I **criteri di valutazione**

## Aggiungere un nuovo progetto

1. Copia il [template-progetto.md](../template-progetto.md)
2. Compila tutte le sezioni
3. Inserisci il file nella cartella `catalogo/`
4. Aggiorna questa tabella
5. Apri una PR con revisione collegiale

---

## Progetti trasversali (pluriennali)

Alcuni progetti si sviluppano su più anni:
- **Digital Portfolio** (anni 1–5): ogni studente mantiene un portfolio personale aggiornato ogni anno
- **Open Source School** (anni 3–5): i team del triennio contribuiscono a un progetto open source dell'istituto
