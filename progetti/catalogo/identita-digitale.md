# Progetto: Identità Digitale

| Campo | Valore |
|-------|--------|
| **Titolo** | Identità Digitale – Il mio portfolio sul web |
| **Anno di corso** | 1° |
| **Durata** | Ottobre – Febbraio (5 mesi) |
| **Team size** | Individuale + peer review |
| **Stato** | `attivo` |
| **Referente** | Docente A041 |

---

## Il Problema

Gli studenti arrivano al primo anno con un'identità digitale che non controllano:
profili social, dati sparsi, nessuna presenza professionale.
Il problema è: **come costruire una presenza digitale consapevole e professionale
che crescerà con loro per tutto il quinquennio e oltre?**

---

## Committente / Stakeholder

- Lo **studente stesso** (il portfolio è suo)
- La **scuola** (vetrina del dipartimento)
- I **futuri datori di lavoro / università**

---

## Output attesi

- [x] Sito web personale pubblicato online (HTML/CSS, GitHub Pages)
- [x] Pagina "Chi sono" con presentazione personale
- [x] Sezione "Progetti" (inizialmente vuota, si riempirà nei 5 anni)
- [x] Riflessione scritta su privacy, sicurezza e identità online
- [x] Repository GitHub personale con storia dei commit

---

## Macro-fasi

| Fase | Descrizione | Durata |
|------|-------------|--------|
| 1. Discovery | Chi sono online? Analisi della propria presenza digitale attuale | 2 sett. |
| 2. Design | Wireframe del sito, raccolta contenuti, scelta della struttura | 1 sett. |
| 3. Build | HTML, CSS, pubblicazione su GitHub Pages | 6 sett. |
| 4. Revisione | Peer review, miglioramenti, test su mobile | 2 sett. |
| 5. Presentazione | Demo alla classe + riflessione scritta | 1 sett. |

---

## Strumenti e tecnologie

- **Sviluppo:** VS Code, HTML/CSS, Git, GitHub
- **AI:** ChatGPT (brainstorming contenuti, correzione testi), GitHub Copilot (HTML/CSS)
- **Collaborazione:** GitHub, peer review in classe
- **Deployment:** GitHub Pages (gratuito, sempre online)

---

## Criteri di valutazione

| Criterio | Peso |
|----------|------|
| Sito funzionante e pubblicato | 30% |
| Qualità dei contenuti e comunicazione | 25% |
| Riflessione su identità digitale e privacy | 20% |
| Uso di Git (storia dei commit) | 15% |
| Uso consapevole dell'AI (documentato) | 10% |

---

## Note

Questo progetto è **pluriennale**: il sito viene aggiornato ogni anno con i nuovi progetti.
Al 5° anno diventa un portfolio professionale completo, usato per il colloquio dell'Esame di Stato.
