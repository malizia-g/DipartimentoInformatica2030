## Progetto: WebApp + IoT Smart School

**Descrizione:** Gli studenti costruiscono un sistema IoT per monitorare la scuola:
temperatura, occupazione aule, qualità dell'aria, con dashboard web in tempo reale.

**Output attesi:**
- Backend REST API (Node.js / Python Flask / Django)
- Frontend web con visualizzazione dati in tempo reale
- Rete di sensori (Arduino/Raspberry) integrati nel sistema
- Documentazione architetturale (diagrammi, API docs)
- Report di sicurezza del sistema

**Ruoli nel team:** Frontend dev, Backend dev, Hardware engineer, DevOps, Documenter
