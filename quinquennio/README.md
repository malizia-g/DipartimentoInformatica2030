# Quinquennio – Struttura Curricolare

Il quinquennio è organizzato **per progetti, non per programmi**.
Ogni anno ha un **tema portante** attorno al quale ruotano le attività didattiche.
Le discipline non scompaiono: forniscono gli strumenti per completare i progetti.

## Filosofia

```
Anno 1 → scoperta
Anno 2 → costruzione
Anno 3 → connessione
Anno 4 → intelligenza
Anno 5 → autonomia
```

### I contenuti NON vengono definiti a priori

I contenuti di ogni disciplina **emergono dai bisogni del progetto**.
Se un progetto richiede le basi di Python → si impara Python.
Se richiede la statistica → si impara la statistica.
Questo garantisce **apprendimento contestualizzato e significativo**.

## Riforma 2025 – note

Con il D.Lgs. 150/2024 (riordino degli istituti tecnici), il quinquennio subisce modifiche
strutturali che entrano in vigore dall'a.s. 2025/2026 per le classi prime:

| Aspetto | Prima della riforma | Dopo la riforma |
|---------|--------------------|--------------------|
| Monte ore annuale | 1.056 h (33h/sett.) | 1.056 h con maggiore flessibilità |
| Quota autonomia | 20% | Fino al 30% |
| PCTO (ex alternanza) | 210 h triennio | Potenziato, anche nel biennio |
| Orientamento | Opzionale | Obbligatorio e trasversale |
| Competenze digitali | Informatica + lab | Trasversali a tutte le discipline |
| Lingua inglese | Standard | Potenziato (possibile CLIL obbligatorio) |
| Flessibilità oraria | Limitata | Maggiore, anche per moduli intensivi |

Questo quinquennio è progettato per sfruttare la **quota di autonomia del 30%**
per consolidare i progetti interdisciplinari e le ore di laboratorio esteso.
La struttura per progetti è allineata con la nuova enfasi su **orientamento al lavoro**
e **competenze trasversali** richieste dalla riforma.
