# DipartimentoInformatica2030

> **Dipartimento di Informatica e Telecomunicazioni – Indirizzo Informatico**
> Istituto Tecnico Industriale · Quinquennio 2025–2030
> Gestito come un progetto software, pianificato con agenti AI.

---

## 🧭 Visione

Questo repository è il **sistema di gestione** del dipartimento di informatica.
Niente più programmi statici: i **contenuti emergono dai progetti**.
Gli **agenti AI** supportano la pianificazione, la revisione e l'aggiornamento continuo del curricolo.

### Principi guida

| Principio | Descrizione |
|-----------|-------------|
| 🗂 **Repository come sistema** | Tutto è versionato: cattedre, progetti, metodologie, valutazioni |
| 🚫 **Niente programmi predefiniti** | I contenuti non vengono elencati a priori, emergono dai progetti |
| 🤖 **AI-first planning** | Gli agenti AI affiancano i docenti nella progettazione didattica |
| 🔁 **Iterazione continua** | Il curricolo si aggiorna come il software: per issue, PR e release |
| 🎯 **Project-based learning** | Ogni anno ruota attorno a progetti reali e significativi |

## 🚀 Come contribuire (workflow del dipartimento)

1. **Apri una issue** per proporre una modifica a un progetto, una cattedra o una metodologia
2. **Crea un branch** con il nome della modifica (es. `feature/progetto-chatbot-3anno`)
3. **Usa un agente AI** della cartella `agenti/` per supportare la progettazione
4. **Apri una Pull Request** con revisione collegiale
5. **Merge e deploy** → il curricolo è aggiornato
